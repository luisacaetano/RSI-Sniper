================================================================
              RSI SNIPER PRO - Instalacao
================================================================

INSTRUCOES:

1. Identifique seu sistema operacional

2. Entre na pasta correspondente:
   - Windows/  -> Para Windows 10/11
   - macOS/    -> Para macOS (Big Sur+)
   - Linux/    -> Para Ubuntu/Debian/Fedora

3. Execute o instalador:
   - Windows: Duplo clique em "Instalar_RSI_Sniper.bat"
   - macOS:   Duplo clique em "Instalar_RSI_Sniper.app"
   - Linux:   Execute "./instalar.sh" no terminal

4. Siga as instrucoes na tela

5. Apos instalar, compile o EA no MetaEditor (F7)


INSTALACAO VIA TERMINAL (Alternativa):

   python3 install_rsi_sniper.py


ARQUIVOS DO ROBO:

   Os arquivos fonte estao na pasta Scripts/:
   - RSI_Sniper.mq5  (Expert Advisor)
   - RSIExport.mqh   (Biblioteca)
   - rsi_panel.py    (Painel de Controle)


PROBLEMAS?

   Leia o arquivo LEIA-ME.txt dentro da pasta do seu SO.
